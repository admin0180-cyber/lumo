Lumo
Sobre o Projeto

O Lumo é uma aplicação web desenvolvida para auxiliar usuários na organização de tarefas e no gerenciamento de atividades do dia a dia. O projeto surgiu com o objetivo de oferecer uma solução simples, moderna e intuitiva para aumentar a produtividade, permitindo que cada usuário acompanhe suas responsabilidades em um único ambiente.

A plataforma conta com um sistema de autenticação que garante segurança e privacidade dos dados, permitindo que cada usuário tenha acesso apenas às suas próprias informações. Após o login, é possível criar, editar, visualizar e remover tarefas, além de definir categorias, prioridades e prazos para uma melhor organização.

O desenvolvimento do projeto foi realizado utilizando FastAPI no backend, responsável pelo processamento das requisições, autenticação e integração com o banco de dados. Para o armazenamento das informações foi utilizado SQLite, enquanto o frontend foi construído com HTML, CSS e JavaScript, proporcionando uma interface responsiva e de fácil utilização.

A arquitetura da aplicação segue o modelo cliente-servidor, promovendo uma separação clara entre a interface visual e as regras de negócio. Essa abordagem facilita a manutenção do sistema, a implementação de novas funcionalidades e futuras evoluções do projeto.

Além de atender aos requisitos acadêmicos propostos, o Lumo representa uma aplicação completa de desenvolvimento Full Stack, reunindo conceitos de banco de dados, APIs REST, autenticação de usuários, segurança da informação e experiência do usuário em uma solução funcional e pronta para expansão.

Objetivo

O principal objetivo do Lumo é fornecer uma ferramenta prática para gerenciamento de tarefas, contribuindo para a organização pessoal e profissional dos usuários por meio de uma experiência simples, segura e eficiente.
